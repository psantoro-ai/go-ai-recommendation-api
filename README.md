Go AI Recommendation API

Simple REST API built with Go that provides content recommendations.

🚀 Features

- RESTful endpoint
- Fast and lightweight
- JSON responses
- Category-based filtering

🛠️ Tech Stack

- Go (Golang)
- net/http

 ▶️ Run

```bash
go run main.go

🔗 Endpoint
GET /recommend?category=ai
Example:
http://localhost:8080/recommend?category=ai
